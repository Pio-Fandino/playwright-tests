# Playwright Tests 🚀

This repository contains automated end-to-end tests built with [Playwright](https://playwright.dev/).  
It is part of my **automation certification and ISTQB preparation journey**, showcasing practical test automation and CI/CD integration.

---

## 📂 Project Structure
- `tests/` → Test files (example, login, homepage, locators)
- `playwright.config.ts` → Playwright configuration
- `.github/workflows/playwright.yml` → GitHub Actions workflow for CI/CD
- `.gitignore` → Ignored files and folders
- `results.xml` → Test results output

---
